This is a simple personal portfolio website built using **HTML** and **basic CSS**. It is designed to present my profile and provide an easy way for visitors to get in touch.

The website currently includes two main pages: a Home page that introduces me and highlights my work, and a Contact page for communication.

* Two-page structure (Home & Contact)
* Easy navigation between pages
* Responsive and beginner-friendly design
